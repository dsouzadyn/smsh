# smsh
A toy shell written in rust. Soley created for the purpose of learning rust, though improvements and suggestions are welcome.

### Requirements
1. rustc v1.29.0
2. cargo v1.29.0

### Running
```
$ cargo run
```

### Building

Unoptimized debug version:
```
$ cargo build
```

Optimized release version:
```
$ cargo build --release
```
